## 🛠️ Design Principles
- **No‑Line Rule**: Avoid 1px borders; rely on color contrast and spacing for separation.
- **Glassmorphism**: Action areas use subtle translucency and backdrop blur to create depth.
- **Tonal Layering**: Layer surfaces using varying tones of the primary palette to convey hierarchy.
